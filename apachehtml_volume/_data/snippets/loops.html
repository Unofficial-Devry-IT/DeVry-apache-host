Loops

Compare how to write a loop in multiple languages

For Loop

When you need to repeat something a finite amount of times, this is the tool to use!
Let's pretend to "spawn" 5 enemies! 


                    # this for loop -- x starts at 0 and ends at 4. Running for a total of 5 times.

# This means we need to add 1 to x for it to be "human-readable"

for x in range(5):

     print(f"Spawning enemy #{x+1}")

For more information how to use range go here 


                    # The "stop" parameter is exclusive, meaning the value you provide will NOT be included.

# So stopping at 6 means 5 is the last number!

for x in range(1, 6):

     print(f"Spawning enemy #{x}")

In languages such as C#, Java, C++, the for loop is constructed as follows.

  • Variable: tracks the current iteration the loop is on
  • Condition: so long as variable's value meets the specified condition -- the body of the loop will execute
  • Expression: what happens to variable at the end of each pass/iteration. Increment by 1? by 2? Decrement by 1? 3?


                        for(variable; condition; expression)

{

    // code that repeats

}

C#


                        // Note -- we can specify the condition of <= which means 5 is included!

for(int i = 1; i <= 5; i++)

{

    Console.WriteLine($"Spawning enemy #{i}");

}

Java

C++


                        // Note -- we can specify the condition of <= which means 5 is included!

for(int i = 1; i <= 5; i++)

For Each Loop

When you need to perform a task over multiple elements in a collection (list, array, etc), a for-each loop will make life a bit easier!

Lets pretend we have an array of groceries. 

"apple", "orange", "banana", "grapefruit"


                        for item in groceries:

     print(f"Picked up {item} out of my shopping cart")

                        

                    


                        foreach(string item in groceries)

{

    Console.WriteLine($"Picked up {item} out of my shopping cart");

}

                        

                    


                        for(String item : groceries)

{

    System.out.println("Picked up " + item + "out of my shopping cart");

}
